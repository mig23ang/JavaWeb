Enviando parametros por url
ejemplo de la url http://localhost:8080/params/string?texto=holaputos

  • /params/string?texto=hola texto hola
  • /params/string?texto=jhon texto jhon
  • /params/mix-params?saludo=hola&numero=7 jhon texto jhon
  • /params/mix-params?saludo=hola&numero=7 jhon texto jhon
